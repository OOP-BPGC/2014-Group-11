Installation Procedure :- 

Our application has a number of dependencies which are listed below with the download url. 
 
1) junit-test-3.8.1     http://central.maven.org/maven2/junit/junit/3.8.1/junit-3.8.1.jar 
2) sqlite-jdbc-3.8.7    https://bitbucket.org/xerial/sqlite-jdbc/downloads/sqlite-jdbc-3.8.7.jar
3) servlet-api.jar (will come automatically with tomcat7) http://www.java2s.com/Code/JarDownload/servlet/servlet-api.jar.zip 
4) tomcat7    http://www.dsgnwrld.com/am/tomcat/tomcat-7/v7.0.57/bin/apache-tomcat-7.0.57.tar.gz         

To build and run our application, you need to have the above dependencies installed and after that follow the given below steps :- 

1) make run :- To run the precompiled binary file of our application. 
2) make all :- To compile the unit tests, source files and run the application. 
3) make ctest :- To only compile the unit tests. 
4) make test :- To only run the already compiled unit tests. 
5) make clean :- To delete the binary files.
